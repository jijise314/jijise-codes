# 최대공약수(GCD), 최소공배수(LCM) 구하기

두 정수 간의 최대공약수와 최소공배수를 계산하기 위해 작성한 코드입니다.

연산 정의
get_gcd -> 유클리드 호제법을 사용하여 두 정수 a와 b의 최대공약수를 계산하여 반환, 음수가 입력되어도 절댓값으로 처리하여 정확한 값을 도출

get_lcm -> 두 정수 a와 b의 최소 공배수를 계산하여 반환

사용법
프로그램을 실행하면 콘솔을 통해 두 개의 정수를 입력받습니다.

EX) 두 정수를 입력하세요 (예: 18 24): 18 24
--결과--
GCD: 6, LCM: 72

잘못된 형식의 문자를 입력할 경우 에러 메시지를 출력하고 프로그램을 종료
