# 분수 계산하기

데이터

num1 (int): 분수를 구성하는 분자

num2 (int): 분수를 구성하는 분모 (0이 될 수 없음)

typedef struct {
    int num1; // 분자
    int num2; // 분모
} Fraction;

연산정의

create_fraction -> 분자와 분모를 가진 새로운 분수 구조체를 생성하여 반환함

reduce -> 최대공약수(GCD)를 이용하여 주어진 분수 f를 기약분수 형태로 약분하여 반환

add -> 두 분수 f1과 f2를 더한 후, 그 결과를 기약분수로 반환

minus -> 첫 번째 분수 f1에서 두 번째 분수 f2를 뺀 후, 그 결과를 기약분수로 반환

multiply -> 두 분수 f1과 f2를 곱한 후, 그 결과를 기약분수로 반환

divide -> 첫 번째 분수 f1을 두 번째 분수 f2로 나눈 후, 그 결과를 기약분수로 반환

scan_fraction -> 사용자로부터 분자/분모 형식으로 입력을 받아 파싱한 뒤, 기약분수 형태로 변환하여 반환

print_fraction -> 주어진 분수 f를 분자/분모 형식의 문자열로 콘솔에 출력
